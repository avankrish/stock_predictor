# Stock Predictor using LSTM Neural Networks

A deep learning-based stock market prediction system using Long Short-Term Memory (LSTM) neural networks for forecasting stock closing prices.

---

## Project Overview

This project focuses on predicting stock market closing prices using historical stock data and deep learning techniques.

The system leverages Long Short-Term Memory (LSTM) neural networks to identify temporal patterns in stock price movements and generate predictive insights for market trend analysis.

The project was developed as part of an undergraduate final year research project and was presented at an academic conference focused on emerging technologies and intelligent systems.

---

## Model Architecture

The project uses a Long Short-Term Memory (LSTM) neural network for sequential time-series prediction.

Workflow:

1. Data collection and preprocessing
2. Feature scaling and sequence generation
3. LSTM model training
4. Prediction of future closing prices
5. Visualization of predicted vs actual prices
